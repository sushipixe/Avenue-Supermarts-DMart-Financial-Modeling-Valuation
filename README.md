## 📈 Project Overview

This Financial Modeling project provides a comprehensive analysis of Avenue Supermarts (DMart) by integrating historical financial data, forecasting assumptions, valuation methodologies, and risk assessment tools. The model evaluates the company's operational performance, profitability, liquidity, and growth prospects while estimating its intrinsic value through DCF and relative valuation techniques.

The project demonstrates practical applications of Corporate Finance, Equity Research, Financial Statement Analysis, Forecasting, and Valuation. It is designed to support data-driven investment decisions and showcase financial modeling skills commonly used in investment banking, equity research, and corporate finance roles.
